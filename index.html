What is PHPush?

PHPush is a collection of PHP scripts which are easily modified, and meant for sending push notifications to your apps that use either Parse or Urban Airship. In other words, they perform cURLs based on functions.

They will continue to be expanded to allow maximum flexibility, and ease of use, and will be tightly-woven with both the Parse REST API and Urban Airship API. They are meant to be hosted on a server or website to allow your project to POST or GET, etc. (based on the script/function) and return a JSON result.

More Info

PHPush scripts are meant to be as easy to use and as versatile as possible, allowing you to send notifications to your app through a wide range of languages and projects.

You can use Ruby, HTML, jQuery and Java. Basically, if the language you are using can perform an HTTP POST request, than you can use this (soon to be) library.

By default, all of these scripts return a JSON response, some simple error messages, such as {"error"="unauthorized"} Some, success messages, such as {"result"="true"}

However, not all of the scripts/functions will return a result in this fashion. In some cases, the JSON response is not generated by the script, it is generated by the Parse or Urban Airship servers.

Although every script returns a JSON response, it is not necessary to use this response.
